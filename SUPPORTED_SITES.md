# AutoPlay Video Control - Supported Streaming Sites

This is a complete list of all streaming platforms supported by AutoPlay Video Control.

Last updated: 2026-01-02

## Streaming Platforms

- **Netflix** - netflix.com
- **YouTube** - youtube.com
- **Disney+** - disneyplus.com
- **HBO Max / Max** - max.com, hbomax.com
- **Prime Video** - primevideo.com, amazon.com
- **Hulu** - hulu.com
- **Crunchyroll** - crunchyroll.com
- **Twitch** - twitch.tv

## Features

✅ Sleep timer with customizable durations  
✅ Auto-pause when timer expires  
✅ On-screen countdown overlay  
✅ Keyboard shortcuts  
✅ Context menu integration  
✅ Works across all tabs  

## Adding More Sites

Want AutoPlay to support your favorite streaming platform? Run the included setup script:

```bash
node add-streaming-site.js
```

Or open an issue on our [GitHub repository](https://github.com/kinect3/AutoPlayVideo) with your request!

## Browser Support

- Google Chrome
- Microsoft Edge
- Brave
- Opera
- Any Chromium-based browser

---

**Note**: This extension respects the native video controls of each platform and does not interfere with DRM or premium content restrictions.
